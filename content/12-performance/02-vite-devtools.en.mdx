  // Refer to the official Vite 8.1.x documentation

## vite-plugin-inspect: plugin-level debugging

`vite-plugin-inspect` is the most useful debugging tool in the Vite ecosystem:

```bash
pnpm add -D vite-plugin-inspect
```

```ts filename="vite.config.ts"
import { defineConfig } from "vite"
import Inspect from "vite-plugin-inspect"

export default defineConfig({
  plugins: [
    Inspect(),
    // ... other plugins
  ],
})
```

After starting the dev server, visit `http://localhost:5173/__inspect`:

### What you can see

1. **Module list**: all loaded modules and their transform times
2. **Transform chain**: which plugins processed each module and how long each took
3. **Code diff**: before-and-after code comparison for each plugin
4. **Plugin graph**: plugin call relationships

### Typical use cases

**Case 1: A specific file loads very slowly**

Sort the module list by time to find the slowest module. Click in to see which plugin is responsible.

**Case 2: A plugin isn't taking effect**

Look for your plugin in the module's transform chain. If it's not in the list:

- Check whether the `apply` configuration is correct
- Check the file filter conditions

**Case 3: Code is being modified unexpectedly**

Use diff mode to see what each plugin changed, and find the plugin modifying content you didn't expect.

## Chrome DevTools Performance

For lower-level performance analysis, use Chrome DevTools directly:

```bash
# Enable the V8 profiler
node --prof $(which vite) dev
node --prof-process isolate-*.log > processed.txt
```

Or use Clinic.js:

```bash
pnpm add -g clinic
clinic doctor -- node ./node_modules/.bin/vite dev
```

## Debugging transform timing

Add timing instrumentation inside your own plugins:

```ts
import type { Plugin } from "vite"

export function timedPlugin(wrappedPlugin: Plugin): Plugin {
  const original = wrappedPlugin.transform

  return {
    ...wrappedPlugin,
    transform(code, id, options) {
      const start = performance.now()
      const result = original?.call(this, code, id, options)
      const elapsed = performance.now() - start

      if (elapsed > 10) {
        console.log(`[${wrappedPlugin.name}] transform ${id}: ${elapsed.toFixed(1)}ms`)
      }

      return result
    },
  }
}
```
